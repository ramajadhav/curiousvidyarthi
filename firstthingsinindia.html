भारतातील पहिल्या विविध घटना आणि गोष्टी

⚫ भारतातील पहिले वर्तमानपत्र - द बेंगॉल गॅझेट ( जेम्स हिक, २९ जानेवारी १७८१)

⚫ भारतातील पहिली टपाल कचेरी - कोलकत्ता (१७२७)

⚫ भारतातील पहिले रेल्वे ( वाफेचे इंजिन ) - मुंबई ते ठाणे ( १६ एप्रिल, १८५३ )

⚫ भारतातील पहिली विजेवरील रेल्वे - मुंबई ते कुर्ला (१९२५)

⚫ भारतातील पहिली भुयारी रेल्वे - मेट्रो रेल्वे, कोलकत्ता

⚫ भारतातील पहिली दुमजली रेल्वे - सिंहगड एक्सप्रेस (मुंबई ते पुणे )

⚫ भारतातील पहिला मूकपट - राजा हरिश्चंद्र (१९१३, दादासाहेब फाळके निर्मित )

⚫ भारतातील पहिला बोलपट - आलमआरा ( १९३१, आर्देशीर इराणी निर्मित )

⚫ भारतातील पहिला मराठी बोलपट - अयोध्येचा राजा ( १९३२, प्रभात फिल्म कंपनीच्या व्ही. शांताराम निर्मित )

⚫ भारतातील पहिला ऍनिमेशन चित्रपट - हनुमान ( दिग्दशर्क - व्ही. जी. सामंत- २००५)

⚫ भारतातील पहिले दूरदर्शन केंद्र - दिल्ली (१९५९) .

⚫ भारतातील पहिले आकाशवाणी केंद्र - मुंबई (१९२७)

⚫ भारतातील पहिले विद्यापीठ - कोलकत्ता विद्यापीठ (१८५७)

⚫ भारताचा पहिला उपग्रह - आर्यभट्टा (१९७५)

⚫ भारतातील पहिला अणुस्फोट - पोखरण ( १८ मे १९७४, राजस्थान)

⚫ भारतातील पहिले क्षेपणास्त्र - पृथ्वी (१९८८)

⚫ भारताचे पहिले लढाऊ विमान - नॅट

⚫ भारताची पहिली अंटार्क्टिका मोहीम - डिसेंबर १९८१ ( मोहीम प्रमुख प्रा. कासीम)

⚫ भारतातील पहिली अणुभट्टी - अप्सरा, तारापूर (१९५६)

⚫ भारतातील पहिली कापड गिरणी - मुंबई (१८५४) .

⚫ भारतातील पहिली ताग गिरणी - कोलकत्ता (१८५५)

⚫ भारतातील पहिला सिमेंट कारखाना - चेन्नई (१९०४)

⚫ भारतातील पहिले जलविद्युत केंद्र - दार्जिलिंग (१८९८)

⚫ भारतातील पहिले पंचतारांकित हॉटेल - ताजमहाल, मुंबई (१९०३)

⚫ भारतातील पहिले व्यापारी विमानोड्डाण - कराची ते मुंबई ( ओक्टोम्बर १९३२)

⚫ भारतातील पहिले राष्ट्रीय उद्यान - जीम कार्बेट राष्ट्रीय उद्यान ( उत्तरांचल, १९३५)

⚫ भारतातील पहिले संग्रहालय - इंडियन म्युसिअम, कोलकाता ( फेब्रुवारी १८१४)

⚫ भारतातील पहिला सहकारी साखर कारखाना - प्रवरानगर ( १९५०, अहमदनगर)

⚫ भारतातील पहिली सहकारी सुतगिरणी - कोल्हापूर जिल्हा शेतकरी विणकर सहकारी संस्था, इचलकरंजी

⚫ भारतातील पंचायत पद्धतीचा स्वीकार करणारे पहिले राज्य - राजस्थान

⚫ भारतातील पहिले शंभर टक्के साक्षर शहर - कोट्टायम ( केरळ )

⚫ भारतातील पहिला शंभर टक्के साक्षर जिल्हा - अर्नाकुलम ( केरळ)

⚫ भारतातील पहिला अणुस्फोट - पोखरण ( १८ मे १९७४, राजस्थान)

⚫ भारतातील नौदलातील पहिली युध्दनौका - विक्रांत

⚫ भारतातील पहिली आण्विक पाणबुडी - आय. एन. एस. चक्र

⚫ भारताचे अंटार्टिकावरील पहिले स्थानक - दक्षिण गंगोत्री ( दुसरे मैत्री )

⚫ भारतातील पहिले मुक्त विद्यापीठ - हैदराबाद, आंध्रप्रदेश ( १९८२)

⚫ अंत्योदय योजना सुरु करणारे भारतातील पहिले राज्य - राजस्थान

⚫ भारतातील राष्ट्रसभेचे पहिले अधिवेशन - गोकुळदास तेजपाल संस्कृत पाठशाळा, मुंबई ( २८ डिसेंबर, १८८५)

⚫ भारतातील हवामानविषयक पहिला उपग्रह - कल्पना-१ ( १२ सप्टेंबर २००३, श्रीहरीकोटा )
